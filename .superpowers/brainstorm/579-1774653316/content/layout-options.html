C. Tabbed Config + Progress Bar

Like A but with a dedicated progress bar section between controls and log. More visual feedback on sync state.

Recommendation: B

  • See both server + client simultaneously
  • Perfect for the common localhost sync case
  • Shared log shows interleaved activity
  • Each side has independent controls

Trade-off

  • Needs wider window than tabbed layout
  • Config panels are more compact (less room per side)
  • Can combine with C's progress bar in the shared area
